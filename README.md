# Lollipop

Por ahora, el único propósito de esta gema es instalar un conjunto de gemas de desarrollo a un proyecto.

El listado completo es:
 * `brakeman`
 * `bundler-audit`
 * `flay`
 * `flog`
 * `guard`
 * `guard-rspec`
 * `guard-rubocop`
 * `guard-shell`
 * `lefthook`
 * `libyear-bundler`
 * `rubocop`
 * `rubocop-performance`
 * `rubocop-rails`
 * `rubocop-rake`
 * `rubocop-rspec`
 * `rubocop-sorbet`
 * `sorbet`
 * `sorbet-progress`
 * `terminal-notifier-guard`
